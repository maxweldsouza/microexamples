Ten Year Course

I've only been programming for the past three years. This is a collection of resources that I have used. You can use them in any order. However I've arranged them so that the more advanced stuff is at the bottom. If you don't want to buy the books, read the first few pages on Amazon first. Buy the books if you like them.

Read Peter Norvig's article

Read this article by Peter Norvig, the director of research at Google. He says how it takes 10 years to get good at anything and it's the same for programming. Teach yourself programming in ten years

Learn to type

You need to know to type if you want to learn Vim, and you definately want to learn vim.

Introduction to Vim

Watch Derek Wyatt's video. Get a taste of the power of Vim. Introduction to Vim

More on Vim

Read this great post on Vim

Richard Feynman on Computing

Richard Feynman's Lecture on Computer Heuristics

Learn how to use vim incrementally

You can't learn all of vim, not in one lifetime. Learn just what you need.

Guy Steele: Growing a language

Guy Steele talks about why Java needs generics and operator overloading. This talk is one of a kind. Watch it to know why. Growing a language

Alan Kay's turing award lecture

Alan Kay talks about keeping things simple, Sketchpad etc..

Alan Kay's talk on OOP

Alan Kay talks on object oriented programming at OOPSLA 1997. The computer revolution hasn't happened yet Greate Quotes:

  • I invented the term Object-Oriented, and I can tell you I did not have C++ in mind.
  • (About html) This has to be one of the worst ideas since MS DOS.

Watch Linus Torvalds' talk on Git

Linus Torvalds talks about Git at Google Linus Torvalds on Git

Do the basic course on Git at Codeschool

Learn the basics of Git. Try Git

Read hackers and painters

Hackers and painters is among the best books I have ever read. It is a collection of essays of Paul Graham. It talks about why web software is great, how to make wealth, and why lisp is so great. Hackers and Painters

Surely You're Joking Mr Feynman

This book isn't about programming but it is a must read. Nobel Prize winning physicist, Richard Feynman shares some of his humourous stories. Surely you're joking Mr Feynman

The inmates are running the asylum

This is a great boook on UI design. The inmates are running the asylum

Richard Feynman on Nanotechnology

Another fantastic lecture by Prof Richard Feynman Richard Feynman's Lecture on Nanotechnology
