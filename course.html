FAQ

Why did you create this site?

I wanted to learn more programming languages. I thought creating this site would help me as well as others learn new languages.

Every other book has examples. Whats new here?

Unlike most other places, I have tried to focus much more on errors. There are several examples here of common mistakes so that you become familiar with common error messages you'll encounter.

Who is this site for?

This site is for people who already know to program and want to learn a new language.

Is this a scientifically proven way to learn new languages?

I happen to be a researcher however I don't know. I have no experience at all in this area. I just created this site know that it is something I would have wanted.

Why are there no explanations?

If you already know programming, explanations just clutter reading material. You can understand enough of the code just by reading the source code and the output.
